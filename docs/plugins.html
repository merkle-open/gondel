Data Plugin - Provide auto binding of data attributes

Hot Plugin - Adds support for hot-module-reloading (hmr) for Gondel components.

https://codesandbox.io/s/github/namics/gondel/tree/master/examples/typescript

Media Queries Plugin - Provide a custom gondel event which will fire once a given media query is met

https://codesandbox.io/s/github/namics/gondel/tree/master/examples/plugin-media-query

React Plugin - Adds support to bootstrap React widgets and apps using Gondel and vice versa

https://codesandbox.io/s/github/namics/gondel/tree/master/examples/gondel-react
